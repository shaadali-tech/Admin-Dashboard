# 🚀 Admin Dashboard

A responsive admin dashboard built using **React** and **Material UI (MUI)**.

## ✨ Features

* Responsive Sidebar (desktop + mobile)
* Dark / Light mode
* Data Table (edit & delete)
* Charts (Recharts)
* Clean UI with MUI

## 🛠 Tech Stack

* React
* Material UI
* MUI DataGrid
* Recharts

## ⚙️ Run Project

```bash
npm install
npm run dev
```
